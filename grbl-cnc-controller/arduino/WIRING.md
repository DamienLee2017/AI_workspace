# Arduino CNC 控制器接线说明

本文档详细介绍 Arduino CNC 控制器各引脚的接线方法，适用于 Arduino Uno、Nano 和 Mega 板。

---

## 硬件准备

### 必需组件

| 组件 | 规格 | 数量 |
|------|------|------|
| Arduino 主控板 | Uno / Nano / Mega | 1 |
| 步进电机驱动 | A4988 / DRV8825 / TMC2208 | 3 |
| 步进电机 | NEMA 17 (1.8°/步) | 3 |
| CNC 屏蔽板 | GRBL 兼容 | 1 (可选) |
| 电源 | 12V/24V DC | 1 |
| 限位开关 | 机械式 (6x) / 光学式 | 6 |

### 推荐组件

- 散热片 (步进驱动)
- 滤波器电容 (100μF + 0.1μF)
- 急停按钮
- 主轴电机 (带 PWM 控制)

### 步进电机连接

```
步进电机 (NEMA 17) - 6 线

        线圈A               线圈B
    ┌────────┐          ┌────────┐
    │ 1A  1B │          │ 2A  2B │
    └───┬────┘          └───┬────┘
        │                    │
        ▼                    ▼
    1A ─── 2A (短接 = 共中心抽头) 或
    1B ─── 2B (短接 = 共中心抽头)
    
    或
    
    1A ─── 1B (开路，只用线圈A)
    2A ─── 2B (开路，只用线圈B)
```

## CNC 屏蔽板连接 (推荐)

使用 CNC 屏蔽板可简化接线，提供更好的保护和稳定性。

### 屏蔽板功能

- 集成所有驱动接口
- 内置上拉电阻
- 电源滤波和保护
- 紧凑的接线端子

### 屏蔽板连接

```
Arduino                        CNC Shield
┌──────────┐                  ┌──────────────┐
│ D2-D7    ├──────────────────┤ X/Y/Z STEP   │
│ D8-D11   ├──────────────────┤ X/Y/Z DIR    │
│ D12      ├──────────────────┤ ENABLE       │
│ D13      ├──────────────────┤ LIMIT/PROBE  │
│ A0       ├──────────────────┤ SPINDLE PWM  │
│          │                  │               │
│ 5V/GND   ├──────────────────┤ 板载电源      │
└──────────┘                  └──────────────┘
```

### 电流调节

使用驱动板上的电位器调节输出电流：

```
公式: I_max = V_ref × 2.5

例如:
- NEMA 17 (1.2A): V_ref ≈ 0.48V
- NEMA 17 (1.5A): V_ref ≈ 0.6V
- NEMA 23 (2.0A): V_ref ≈ 0.8V
```

## 常见问题

### 1. 步进电机抖动不转

- 检查驱动电流设置（太低）
- 检查步进电机线圈连接
- 确认 ENABLE 引脚电平正确
- 检查电源电压是否充足

### 2. 电机转向错误

- 交换同一驱动器的 DIR 引脚极性
- 或交换电机线圈 A/B

### 3. 限位开关失效

- 检查上拉电阻是否连接
- 确认开关类型（常开/常闭）
- 检查引脚定义与代码一致

### 4. 驱动板过热

- 增加散热片
- 检查电流设置是否过高
- 改善通风条件

## 安全注意事项

⚠️ **重要警告**

1. **电源极性**: 确保电源正负极连接正确，反接会损坏硬件
2. **电压匹配**: 确认驱动板和电机电压要求匹配
3. **电流限制**: 切勿超过步进电机额定电流
4. **热保护**: 确保驱动板散热良好
5. **接地**: 所有 GND 必须连接在一起形成共地
6. **上电顺序**: 先连接所有线缆，再打开电源
7. **断电操作**: 修改接线前务必关闭电源

---

## 接线检查清单

在通电前请确认以下各项：

- [ ] 电源电压正确 (12V/24V)
- [ ] 驱动板电源连接正确
- [ ] Arduino 与驱动板共地
- [ ] 所有 STEP/DIR/ENABLE 连接正确
- [ ] 步进电机线圈连接正确
- [ ] 限位开关上拉电阻已连接
- [ ] 无裸露的短路导线
- [ ] 驱动板电位器已调节
