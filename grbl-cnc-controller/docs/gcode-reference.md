## G代码 (G-Codes)

### 运动命令

| G代码 | 描述 | 参数 |
|-------|------|------|
| G0 | 快速定位（空走） | X, Y, Z: 目标坐标 |
| G1 | 线性进给（切削） | X, Y, Z, F: 目标坐标和进给速度 |
| G2 | 顺时针圆弧 | X, Y, Z, I, J, F: 终点和圆心偏移 |
| G3 | 逆时针圆弧 | X, Y, Z, I, J, F: 终点和圆心偏移 |
| G4 | 延时暂停 | P: 延时时间（毫秒） |
| G10 L2 | 设置工件坐标系 | P: 坐标系编号, X/Y/Z: 偏移值 |
| G10 L20 | 设置工件坐标系（计算） | P: 坐标系编号, X/Y/Z: 坐标值 |
| G17 | 选择XY平面 | - |
| G18 | 选择XZ平面 | - |
| G19 | 选择YZ平面 | - |
| G20 | 英制单位（英寸） | - |
| G21 | 公制单位（毫米） | - |
| G28 | 返回参考点 | X, Y, Z: 中间点坐标（可选） |
| G28.1 | 设置参考点 | - |
| G30 | 返回第二参考点 | X, Y, Z: 中间点坐标（可选） |
| G30.1 | 设置第二参考点 | - |
| G38.2 | 探针接近（触发） | X, Y, Z: 目标坐标 |
| G38.3 | 探针接近（未触发） | X, Y, Z: 目标坐标 |
| G38.4 | 探针接近（未触发） | X, Y, Z: 目标坐标 |
| G38.5 | 探针接近（触发） | X, Y, Z: 目标坐标 |
| G53 | 机床坐标系定位 | 适用于G0/G1/G28等 |
| G54 | 选择坐标系1 | - |
| G55 | 选择坐标系2 | - |
| G56 | 选择坐标系3 | - |
| G57 | 选择坐标系4 | - |
| G58 | 选择坐标系5 | - |
| G59 | 选择坐标系6 | - |
| G59.1 | 选择坐标系7 | - |
| G59.2 | 选择坐标系8 | - |
| G59.3 | 选择坐标系9 | - |
| G80 | 取消模态动作 | - |
| G90 | 绝对编程 | - |
| G90.1 | 绝对编程（IJK） | - |
| G91 | 增量编程 | - |
| G91.1 | 增量编程（IJK） | - |
| G92 | 坐标系偏移 | X, Y, Z: 偏移值 |
| G92.1 | 清除坐标系偏移 | - |
| G92.2 | 暂时清除偏移 | - |
| G92.3 | 恢复偏移 | - |
| G93 | 进给速度模式：倒数时间 | - |
| G94 | 进给速度模式：每分钟进给 | - |
| G95 | 进给速度模式：每转进给 | - |
| G96 | 恒定表面速度 | S: 速度值 |
| G97 | 取消恒定表面速度 | - |

### 详细说明

#### G0 - 快速定位
```
G0 X10 Y10 Z5
```
快速移动到指定位置，不进行切削。运动速度由系统最大速度决定。

#### G1 - 线性进给
```
G1 X50 Y30 Z-2 F200
```
以指定进给速度（F200 mm/min）移动到目标位置，用于切削加工。

#### G2/G3 - 圆弧运动
```
G2 X30 Y20 I10 J0 F100  ; 顺时针圆弧到(30,20)，圆心相对起点偏移(10,0)
G3 X30 Y20 I10 J0 F100  ; 逆时针圆弧到(30,20)
```
- G2: 顺时针圆弧
- G3: 逆时针圆弧
- I: 圆心相对于起点的X偏移
- J: 圆心相对于起点的Y偏移
- K: 圆心相对于起点的Z偏移（用于螺旋）

#### G4 - 延时
```
G4 P500  ; 延时500毫秒
```

#### G28/G30 - 返回参考点
```
G28      ; 直接返回参考点
G28 X10 Y10  ; 经过中间点(10,10)后返回参考点
G30      ; 返回第二参考点
```

#### G38.x - 探针功能
```
G38.2 Z-10 F50  ; 向Z负方向探查，碰到传感器后停止
```
用于工件找零、对齐和高度测量。

## M代码 (M-Codes)

| M代码 | 描述 |
|-------|------|
| M0 | 程序停止（暂停） |
| M1 | 选择性程序停止 |
| M2 | 程序结束 |
| M30 | 程序结束并返回开头 |
| M3 | 主轴正转 | S: 转速 |
| M4 | 主轴反转 | S: 转速 |
| M5 | 主轴停止 |
| M6 | 换刀（GRBL不执行，需要外部处理） |
| M7 | 雾冷却液开启 |
| M8 | 冷却液开启 |
| M9 | 冷却液关闭 |
| M30 | 程序结束 |

### 详细说明

#### M3/M4 - 主轴控制
```
M3 S10000  ; 主轴正转，转速10000 RPM
M4 S8000   ; 主轴反转，转速8000 RPM
```

#### M7/M8/M9 - 冷却液控制
```
M7  ; 雾冷却液开启
M8  ; 冷却液开启
M9  ; 所有冷却液关闭
```

### 查看和设置参数
```
$  ; 查看所有GRBL设置
$$ ; 查看所有参数
$123=250  ; 设置X轴归零速度为250 mm/min
```

### 实时命令

| 命令 | 描述 |
|------|------|
| Ctrl-X | 重置GRBL |
| Ctrl-? | 暂停进给保持 |
| Ctrl-~ | 恢复运行 |
| Ctrl-C | 取消当前行 |
| ? | 实时状态报告 |
| ~ | 恢复运行 |
| ! | 进给暂停 |
| ? | 状态查询 |

### 状态查询响应 (?)

```
<Idle|WPos:0.000,0.000,0.000|Bf:15,127|FS:0,0|Pn:X|Yz>
```

- **Idle/Run/Hold**: 运行状态
- **WPos**: 工件坐标系位置
- **MPos**: 机床坐标系位置
- **Bf**: 缓冲区和可用字节
- **FS**: 进给速度和主轴速度
- **Pn**: 引脚状态（限位开关等）

### 其他控制命令

| 命令 | 描述 |
|------|------|
| $I | 查看构建信息 |
| $N | 查看启动代码 |
| $N0=... | 设置启动代码 |
| $C | 切换G54/G55等坐标系 |
| $X | 解锁报警 |
| $H | 执行归零循环 |
| $J=... | 运行时 Jog 命令 |
| $SLP | 进入睡眠模式 |
| $RST=$ | 恢复默认设置 |
| $RST=# | 清除坐标系统 |
| $RST=* | 恢复默认设置并清除 |

## G代码格式说明

### 基本格式
```
N001 G1 X10.5 Y20.0 F200
```

- **N**: 行号（可选）
- **G1**: G代码命令
- **X10.5 Y20.0**: 坐标参数
- **F200**: 进给速度

### 模态组

GRBL使用模态组来组织命令，同一组的命令会互相替代：

| 模态组 | 包含命令 |
|--------|----------|
| 运动 | G0, G1, G2, G3, G38.2, G38.3, G38.4, G38.5, G80 |
| 坐标系 | G54, G55, G56, G57, G58, G59, G59.1, G59.2, G59.3 |
| 平面选择 | G17, G18, G19 |
| 距离模式 | G90, G91 |
| 弧距离模式 | G90.1, G91.1 |
| 速度模式 | G93, G94 |
| 单位 | G20, G21 |
| 刀具补偿 | G40 |
| 行程补偿 | G98, G99 |
| 主轴 | M3, M4, M5 |
| 冷却液 | M7, M8, M9 |
| 覆盖 | M48, M49 |

### 注释和消息

```
(注释内容)      ; 行内注释
; 纯注释行
```

## G代码示例

### 示例1: 简单的矩形切削
```gcode
G21                 ; 使用公制单位（毫米）
G90                 ; 绝对坐标
G54                 ; 选择坐标系1

; 移动到起始点
G0 X0 Y0 Z5

; 下刀
G1 Z-2 F100

; 切削矩形
G1 X50 F200
G1 Y30
G1 X0
G1 Y0

; 抬刀
G0 Z5

M5                  ; 停止主轴
```

### 示例2: 圆形切削
```gcode
G21                 ; 公制单位
G90                 ; 绝对坐标

G0 X0 Y0 Z5         ; 移动到圆心上方
G1 Z-3 F50          ; 下刀

G2 X25 Y0 I0 J0 F100    ; 顺时针整圆

G0 Z5               ; 抬刀
M5
```

### 示例3: 使用子程序和循环
```gcode
O100 (主程序)
G21 G90
G54

; 切削3个相同的矩形
G0 X0 Y0 Z5
G1 Z-2 F100

M98 P101 L3         ; 调用子程序O101，执行3次

G0 Z10
M30

O101 (矩形子程序)
G1 X30 F200
G1 Y20
G1 X0
G1 Y0
M99                 ; 返回主程序
```

### 示例4: 钻孔循环
```gcode
G21 G90
G54

G81 R5 Z-10 F50     ; 钻孔循环：快速到R点，下刀到Z-10
X10 Y10             ; 钻孔1
X20 Y10             ; 钻孔2
X30 Y10             ; 钻孔3
G80                 ; 取消钻孔循环

M30
```

### 示例5: 带探针找零
```gcode
G21 G90
G54

; Z轴探针找零
G38.2 Z-50 F50      ; 探针Z轴
G92 Z0              ; 设置Z=0
G0 Z5               ; 退回到安全高度

; X轴探针找零
G38.2 X-50 F50      ; 探针X轴
G92 X0              ; 设置X=0
G0 X5               ; 退回到安全高度

; Y轴探针找零
G38.2 Y-50 F50      ; 探针Y轴
G92 Y0              ; 设置Y=0
G0 Y5               ; 退回到安全高度

M30
```

### 示例6: 使用G10设置工件坐标系
```gcode
G21 G90

; 在当前位置设置G54原点
G10 L2 P1 X0 Y0 Z0

; 设置G55原点（工作区左下角偏移）
G10 L2 P2 X-50 Y-30 Z0

M30
```

### 示例7: 激光雕刻图案
```gcode
G21 G90
G54

; 激光功率设置（根据实际调整）
M3 S0

G0 X0 Y0
M3 S255

; 简单的锯齿扫描图案
G1 Y0 F1000
G1 X10 S255
G1 Y5 S0
G1 X20 S255
G1 Y10 S0
G1 X30 S255
; ... 更多线条

G0 X0 Y0
M5
```

## 故障排除

### 常见错误代码

| 错误码 | 描述 | 解决方案 |
|--------|------|----------|
| 1 | G代码命令值无效 | 检查G代码格式 |
| 2 | 单位转换失败 | 检查坐标值 |
| 3 | 负目标位置 | 检查坐标系设置 |
| 4 | 超出软限制 | 增大行程或检查原点 |
| 5 | 无效的组命令 | 检查G代码顺序 |
| 6 | 单位模式未设定 | 添加G20/G21 |
| 7 | 超出安全行程 | 检查G28/G30设置 |
| 8 | 主轴转速为0 | 设置S参数 |
| 9 | 超过最大主轴转速 | 调整$S最大值 |
| 10 | 软限位触发 | 归位或检查行程 |
| 11 | 硬限位触发 | 检查限位开关 |
| 12 | 探针失败 | 检查探针连接 |
| 13 | 引脚读取失败 | 检查接线 |

### 报警代码

| 报警码 | 描述 |
|--------|------|
| 1 | 硬限位触发 |
| 2 | 软件限位触发 |
| 3 | 归零失败 |
| 4 | 归零针脚触发 |
| 5 | 探针失败 |
| 6 | 指令读取失败 |
| 7 | 坐标系统无效 |
| 8 | 孤立G10命令 |
| 9 | 超出X轴行程 |
| 10 | 超出Y轴行程 |
