### Arduino 详细接线步骤

#### 1. 电源接线

```
步骤 1: 电源正极 (+) 连接驱动板 VMOT
步骤 2: 电源负极 (-) 连接驱动板 GND
步骤 3: 在电源输入端并联 100μF 电容（注意极性）
```

**警告**: 确保电源电压与电机和驱动板规格匹配（A4988 最大 35V，DRV8825 最大 45V）

#### 2. 步进电机接线

每个轴（X/Y/Z）需要独立驱动板：

| Arduino 引脚 | A4988 引脚 | 功能 |
|-------------|-----------|------|
| D2 (X) / D3 (Y) / D4 (Z) | STEP | 脉冲输入 |
| D5 (X) / D6 (Y) / D7 (Z) | DIR | 方向控制 |
| D8 (X) / D9 (Y) / D10 (Z) | ENABLE | 使能（低电平有效） |

#### 3. 限位开关接线

| Arduino 引脚 | 限位开关 | 说明 |
|-------------|---------|------|
| D9 | X- 限位 | X 轴负向限位 |
| D10 | X+ 限位 | X 轴正向限位 |
| D11 | Y- 限位 | Y 轴负向限位 |
| D12 | Y+ 限位 | Y 轴正向限位 |
| D13 | Z- 限位 | Z 轴负向限位 |
| A0 | Z+ 限位 | Z 轴正向限位 |

#### 4. 主轴 PWM 控制接线

| Arduino 引脚 | IRF520 模块 | 说明 |
|-------------|------------|------|
| D11 | PWM 输入 | 主轴转速控制 |
| VIN | VCC (5V) | 驱动板电源 |
| GND | GND | 共地 |

| 冷却液 | GPIO 14 | 冷却液控制 |
| WiFi LED | GPIO 15 | 状态指示灯 |

#### 2. 电源系统接线

```
24V 电源 ──┬──→ TMC2209/驱动板 VMOT
           │
           ├──→ 主轴模块 V+
           │
           └──→ 24V→5V 降压模块 ──→ ESP32-S3 Vin
                                              │
           ┌─────────────────────────────────┘
           ▼
        ESP32-S3 3.3V ──→ 驱动板 VCC、限位开关 VCC
           │
           ▼
        所有 GND 连接在一起（共地）
```

#### 3. TMC2209 驱动板接线

TMC2209 提供更安静的运行环境，推荐用于需要低噪音的应用：

| TMC2209 引脚 | ESP32-S3 GPIO | 连接说明 |
|-------------|--------------|---------|
| STEP | 对应 GPIO | 需要 3.3V 信号，可直接连接 |
| DIR | 对应 GPIO | 方向控制信号 |
| EN | 对应 GPIO | 使能信号（低电平启用） |
| VM | 24V | 电机电源 |
| GND | GND | 电源和信号共地 |
| 1A, 1B | 电机 | 连接步进电机线圈 A |
| 2A, 2B | 电机 | 连接步进电机线圈 B |

#### 4. 限位开关接线（光耦隔离）

| ESP32-S3 GPIO | 限位开关端子 | 说明 |
|--------------|------------|------|
| GPIO 11 | X LIM SWITCH | X 轴限位 |
| GPIO 12 | Y LIM SWITCH | Y 轴限位 |
| GPIO 13 | Z LIM SWITCH | Z 轴限位 |
| 3.3V | VCC (限位开关) | 供电 |
| GND | GND (限位开关) | 共地 |

#### 5. WiFi 天线连接

```
ESP32-S3 u.FL 接口 ──→ u.FL 转 SMA 线 ──→ 2.4GHz 天线
```

**注意**: 如果 ESP32-S3 板载天线，需确保天线区域远离金属外壳

## 接线注意事项

### 安全规范

1. **电源安全**
   - 接线前确保电源已关闭
   - 确认输入电压与设备规格匹配
   - 在电源输入端安装保险丝（建议 5A）
   - 使用合适的线径（建议 18AWG 以上用于电源线）

2. **防静电措施**
   - 操作前释放身体静电
   - 避免触摸集成电路引脚
   - 使用防静电手环（推荐）

3. **散热管理**
   - 确保驱动芯片安装散热片
   - 保持良好通风环境
   - 温度超过 60°C 时考虑增加风扇

### 接线质量

1. **连接可靠性**
   - 使用螺丝端子确保牢固连接
   - 杜邦线仅用于临时测试
   - 焊锡连接点确保光滑无虚焊

2. **信号完整性**
   - 电机驱动信号线尽量短
   - 避免信号线与电源线平行走线
   - 敏感信号使用屏蔽线

3. **标识建议**
   - 每个接线点使用标签标识
   - 绘制接线图存档
   - 保留接线照片

### 接地处理

```
正确的接地布局:

主电源负极 ──────────┬──→ 驱动板 GND
                      │
                      ├──→ ESP32/Arduino GND
                      │
                      ├──→ 限位开关 GND
                      │
                      └──→ 主轴模块 GND
```

## 常见问题排查

### 问题 1: 电机不转动

**检查步骤:**
1. 测量驱动板输入电压是否正常（12V/24V）
2. 检查 STEP 信号是否有脉冲输出（用万用表或示波器）
3. 确认 ENABLE 引脚电平正确（A4988 低电平使能）
4. 检查电机线圈是否开路（测量电机接线端子电阻）
5. 检查驱动板是否过热保护

### 问题 2: 电机抖动但不转动

**可能原因:**
1. 脉冲频率过高，尝试降低速度
2. 驱动电流设置过低（调节驱动板电位器）
3. 电机线圈接线错误（检查相序）
4. 电源功率不足

### 问题 3: 限位开关不工作

**排查方法:**
1. 检查限位开关类型（常开/常闭）
2. 测量限位开关输出信号
3. 检查 GRBL/$5 设置（限位开关电平）
4. 检查接线是否松动

### 问题 4: ESP32 WiFi 连接不稳定

**解决方案:**
1. 确保 WiFi 天线正确连接
2. 减少与路由器之间的距离
3. 避免周围强干扰源
4. 检查供电稳定性（WiFi 模块电流峰值可达 500mA）

### 问题 5: 主轴转速异常

**检查项目:**
1. PWM 信号频率是否正确（通常 1-30kHz）
2. 检查主轴控制模块是否损坏
3. 测量 PWM 占空比与电压对应关系
4. 确认主轴电机规格与控制器匹配
