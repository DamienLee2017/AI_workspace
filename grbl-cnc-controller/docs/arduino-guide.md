## 硬件准备

### 必需的电子元件

| 元件 | 推荐型号 | 数量 | 备注 |
|------|----------|------|------|
| 主控制器 | Arduino Uno / Nano | 1 | Uno 稳定性更好 |
| 步进电机驱动 | A4988 / DRV8825 | 3 | DRV8825 噪音更低 |
| 步进电机 | NEMA 17 (1.8°步角) | 3 | 扭矩根据负载选择 |
| CNC 屏蔽板 | GRBL 专用 | 1 | 可选，提升稳定性 |
| 电源 | 12V/24V DC | 1 | 根据电机规格选择 |
| 限位开关 | 机械微动开关 | 3-6 | 建议使用带轴承的 |

### 推荐的 CNC 组件

```
├── 3x NEMA 17 步进电机 (建议 1.5A-2.0A 扭矩)
├── 3x 步进电机驱动模块 (A4988 或 DRV8825)
├── 1x Arduino CNC Shield v3.0 或 equivalent
├── 1x 12V/24V 直流电源 (根据电机功率计算)
├── 3-6x 限位开关 (机械式或光学式)
├── 1x 主轴电机 (电钻或专用主轴)
├── 连接线材和散热片
```

### 工具准备

- 螺丝刀套装
- 焊接工具（如果需要焊接）
- 万用表
- USB 数据线
- 剥线钳

### 步进电机连接

#### A4988 驱动模块设置

1. **调节参考电压 (Vref)**

   ```
   Vref = Imax × 8 × Rs
   
   其中:
   - Imax = 步进电机额定电流 (A)
   - Rs = 传感电阻值 (通常为 0.05Ω 或 0.1Ω)
   
   示例:
   - 使用 1.5A 电机, Rs = 0.1Ω
   - Vref = 1.5 × 8 × 0.1 = 1.2V
   ```

2. **MS1/MS2/MS3 细分设置**

   | MS1 | MS2 | MS3 | 细分 | 步数/圈 (200步电机) |
   |-----|-----|-----|------|---------------------|
   | L   | L   | L   | 全步 | 200                 |
   | H   | L   | L   | 半步 | 400                 |
   | L   | H   | L   | 1/4  | 800                 |
   | H   | H   | L   | 1/8  | 1600                |
   | H   | H   | H   | 1/16 | 3200                |

3. **驱动模块安装方向**

   ```
   注意: 确保驱动模块方向正确
   - EN 引脚朝外
   - 散热片朝上
   - 模块底部不要接触金属外壳
   ```

#### 步进电机接线

标准 4 线 NEMA 17 步进电机接线：

```
电机线颜色代码 (常见):

┌──────────────────────────────────────────────────┐
│  双极性步进电机                                   │
│                                                   │
│  A+  ── 蓝/绿 (绕组1)                             │
│  A-  ── 红/灰 (绕组1)                             │
│  B+  ── 橙/黄 (绕组2)                             │
│  B-  ── 黄/白 (绕组2)                             │
│                                                   │
│  相邻两对是同一个绕组                             │
└──────────────────────────────────────────────────┘
```

**连接步骤**:

1. 识别步进电机的两组线圈（使用万用表测量通断）
2. 将第一组线圈连接到 A+ 和 A-
3. 将第二组线圈连接到 B+ 和 B-
4. 确保方向一致（A+ 和 B+ 不能互换）

### 限位开关连接

```
限位开关类型:

1. 机械开关 (推荐入门)
   - 简单可靠
   - 需要消抖
   
2. 光学开关 (更精确)
   - 无接触磨损
   - 需要注意极性
```

**接线方法**:

```
Arduino CNC Shield 限位接口:

┌─────────────────────────────────┐
│  X-  │  X+  │  Y-  │  Y+  │  Z- │
│  (GND)                         │
└─────────────────────────────────┘

机械开关接线:
   ───┤  ├──── GND
       │      │
      COM    NC
```

**注意**: 如果使用常开 (NO) 开关，需要启用软件上拉电阻。

### 电源连接

```
⚠️ 警告: 确保电源极性正确！

Arduino CNC Shield 电源:
┌─────────────────────────────────┐
│  VMOT+  │  VMOT-  │  GND      │
│   (+)   │   (-)   │  (逻辑地) │
└─────────────────────────────────┘

电压选择:
- 12V: 适用于 NEMA 17 小型电机
- 24V: 推荐用于更大电机或更高速运行
- 绝对不要超过 36V
```

## 软件安装

### 1. Arduino IDE 安装

1. 下载 Arduino IDE
   - 官网: https://www.arduino.cc/en/software
   - 选择 Windows / macOS / Linux 版本

2. 安装驱动程序
   - Windows: 自动安装或手动安装 CH340/FTDI 驱动
   - macOS: 可能需要安装 FTDI USB 驱动

### 2. Grbl 库安装

1. 打开 Arduino IDE
2. 选择 **项目 → 加载库 → 管理库**
3. 搜索 "Grbl"
4. 安装 **Grbl by Skova** 或最新版本

### 3. GrblPanel 下载

GrblPanel 是一个流行的 GRBL 控制器软件：

1. 下载地址: https://github.com/GrblPanel/GrblPanel/releases
2. 选择最新版本的 Windows 安装包
3. 解压到常用位置

## 固件上传

### 使用 Arduino IDE 上传

**方法一: 使用库中的示例**

1. 打开 Arduino IDE
2. 选择 **文件 → 示例 → Grbl → GrblUpload**
3. 选择正确的板卡和端口
4. 点击上传按钮

**方法二: 手动添加 Grbl 固件**

1. 下载 GRBL 固件源码: https://github.com/grbl/grbl
2. 解压后，将 `grbl` 文件夹复制到 Arduino libraries 目录
3. 重启 Arduino IDE
4. 创建新项目，添加头文件:

```cpp
#include <grbl.h>

void setup() {
  grbl.init();
}

void loop() {
  grbl.process();
}
```

### 上传后验证

1. 打开串口监视器 (9600 波特率)
2. 按下 Arduino 的复位按钮
3. 应该看到 GRBL 启动信息:

```
Grbl 1.1h ['$' for help]
```

## 基本操作流程

### 1. 开机检查

```
步骤:
1. 检查所有接线是否牢固
2. 确认电源电压正确
3. 先开控制器电源，后开主轴电源
4. 观察各指示灯状态
```

### 2. 连接软件

```
1. 打开 GrblPanel 或其他控制器软件
2. 选择正确的串口 (通常为 COM3-COM5)
3. 设置波特率: 115200
4. 点击连接
5. 确认连接成功
```

### 3. 归零操作 (Homing)

归零是确定工作坐标系原点的过程：

```
操作步骤:
1. 点击 "Home" 按钮
2. 等待三轴依次移动到限位开关位置
3. 限位开关触发后，轴会缓慢退回
4. 归零完成，坐标归零

注意:
- 首次使用必须进行归零
- 归零前确保限位开关工作正常
```

### 4. 设置工件原点

工件原点是加工程序的起点：

```
方法一: 使用控制器软件
1. 手动移动刀具到工件待加工位置
2. 点击 "Set Zero" 或类似按钮
3. 选择设置的轴 (X/Y/Z)

方法二: 使用 G 代码
G10 L20 P1 X0 Y0 Z0  (设置当前点为工件原点)
```

### 5. 运行加工程序

```
步骤:
1. 加载 G 代码文件 (.nc, .gcode, .txt)
2. 检查预览是否正确
3. 确保安全距离足够
4. 点击 "Run" 开始加工
5. 监控运行状态
6. 完成后检查成品
```

## G 代码入门

### 基础运动指令

#### G0 - 快速定位

```gcode
; 快速移动到 X=50, Y=30 位置
G0 X50 Y30

; 快速移动到原点
G0 X0 Y0 Z0

; 只移动 X 轴
G0 X100
```

#### G1 - 线性进给 (切削)

```gcode
; 以 100mm/min 进给率移动到 X=50, Y=30
G1 X50 Y30 F100

; 从当前位置移动到 Z=-5
G1 Z-5 F50

; 完整的切削移动
G1 X0 Y0 Z-1 F100
```

#### G2/G3 - 圆弧插补

```gcode
; G2: 顺时针圆弧
G2 X50 Y25 I25 J0 F100

; G3: 逆时针圆弧
G3 X50 Y75 I0 J25 F100

; 参数说明:
; X, Y: 目标点坐标
; I: 圆心相对起点的 X 增量
; J: 圆心相对起点的 Y 增量
```

### 坐标系指令

```gcode
; 绝对坐标模式 (默认)
G90

; 相对坐标模式 (增量模式)
G91
G1 X10 Y10 F100  ; 从当前位置移动 10mm
G1 X10 Y10 F100  ; 再移动 10mm

; 返回机械原点
G28

; 返回到第二原点
G30
```

### 主轴控制

```gcode
; 主轴正转 (转速 1000 RPM)
M3 S1000

; 主轴反转
M4 S1000

; 主轴停止
M5

; 注意: S 参数指定转速 (根据主轴规格设置)
```

### 程序示例

简单的矩形雕刻程序：

```gcode
; 文件名: rectangle.nc
; 功能: 雕刻一个 50x30mm 的矩形

; 初始化
G21        ; 公制单位 (mm)
G90        ; 绝对坐标
G28        ; 归零

; 移动到起点 (上方)
G0 Z5      ; 抬刀
G0 X0 Y0   ; 移动到原点

; 开始加工
M3 S3000   ; 启动主轴

G1 Z-1 F50 ; 下刀
G1 X50 F100 ; 沿 X 轴切削
G1 Y30     ; 沿 Y 轴切削
G1 X0      ; 回到 X 原点
G1 Y0      ; 回到 Y 原点

G0 Z5      ; 抬刀
M5         ; 停止主轴

; 程序结束
M30
```

## 限位开关设置

### 硬件连接检查

```
限位开关测试步骤:
1. 打开 GrblPanel
2. 打开命令输入窗口
3. 输入 $$ 查看当前设置
4. 找到 $5 和 $21 参数

$5=0  (限位开关电平: 0=常开, 1=常闭)
$21=0 (限位开关使能: 0=关闭, 1=开启)
```

### 软件配置

```gcode
; 查看所有设置
$$

; 设置限位开关为常开 (HIGH触发)
$5=0

; 启用限位开关
$21=1

; 禁用限位开关 (调试时使用)
$21=0

; 保存设置
```

### 限位触发后的处理

```
限位触发时的现象:
1. 运动立即停止
2. 发出警报 (Alarm)
3. 状态栏显示 "Alarm: Hard limit"

解除步骤:
1. 将轴手动移远离限位开关
2. 在软件中点击 "Unlock" 或发送 $X
3. 系统恢复正常
```

## 校准与调优

### 步进电机校准

#### 计算脉冲数

```
公式:
脉冲数/毫米 = (步进电机步数/圈) × (细分倍数) / (丝杆导程)

示例:
- 步进电机: 200 步/圈
- 驱动细分: 1/16 (MS1=1, MS2=1, MS3=1)
- 丝杆导程: 8mm/圈

计算:
脉冲数/毫米 = 200 × 16 / 8 = 400 脉冲/mm
```

#### GRBL 参数设置

```gcode
; 设置 X 轴脉冲数 (每毫米)
$100 = 400

; 设置 Y 轴脉冲数
$101 = 400

; 设置 Z 轴脉冲数 (注意方向)
$102 = 400

; 查看当前脉冲设置
$100
$101
$102
```

### 进给率校准

```
校准方法:
1. 设置目标进给率 $110 = 1000 (1000mm/min)
2. 运行测试移动 100mm
3. 测量实际移动距离
4. 计算误差百分比
5. 调整 $110 值补偿误差

公式:
新进给率 = 原进给率 × (实测距离 / 目标距离)
```

### 加减速设置

```gcode
; 最大进给率 (mm/min)
$110 = 1000

; 最大快速移动速率
$111 = 2000

; 最大 Z 轴进给率
$112 = 500

; 默认加速度 (mm/s²)
$120 = 50

; X 轴加速度
$121 = 50

; Y 轴加速度
$122 = 50

; Z 轴加速度
$123 = 20

; 最大追及速度
$130 = 200  ; X 轴行程限制
$131 = 200  ; Y 轴行程限制
$132 = 100  ; Z 轴行程限制
```

## 常见问题解决

### 1. 步进电机不转

```
可能原因:
□ 驱动模块未通电
□ 接线错误或松动
□ 驱动模块损坏
□ 电机线圈短路

解决方法:
1. 检查 VMOT 电源电压 (12-36V)
2. 检查 ENABLE 引脚状态
3. 重新接线，确保线圈连接正确
4. 测试单个电机排除故障
```

### 2. 步进电机抖动

```
可能原因:
□ 电源电压不足
□ 驱动电流设置过低
□ 脉冲频率过高
□ 电机与驱动器不匹配

解决方法:
1. 提高电源电压
2. 调整 Vref 增加驱动电流
3. 降低进给率
4. 检查电机规格匹配
```

### 3. 限位开关误触发

```
可能原因:
□ 开关类型设置错误 ($5)
□ 机械开关抖动
□ 接线过长导致干扰
□ 电压过低

解决方法:
1. 正确设置 $5 参数
2. 添加消抖电路或软件滤波
3. 使用屏蔽线
4. 检查接线
```

### 4. 串口连接失败

```
可能原因:
□ 驱动程序未安装
□ 端口被占用
□ USB 线过长或质量差
□ Arduino 固件问题

解决方法:
1. 安装正确的 USB 驱动 (CH340/FTDI)
2. 关闭其他占用串口的程序
3. 更换 USB 线
4. 重新上传固件
```

### 5. G 代码无法识别

```
可能原因:
□ G 代码格式错误
□ 控制器不支持该指令
□ 坐标超出行程范围

解决方法:
1. 检查 G 代码语法
2. 查看 GRBL 支持的指令列表
3. 确保坐标在机器行程内
4. 简化 G 代码测试
```

## 安全注意事项

### 电气安全

```
⚠️ 重要警告:

1. 电源
   - 使用稳定的直流电源
   - 电压不要超过额定值
   - 做好防短路措施

2. 接地
   - 电机驱动器可靠接地
   - 避免电磁干扰

3. 散热
   - 驱动模块安装散热片
   - 保持通风良好
```

### 机械安全

```
1. 防护措施
   - 佩戴护目镜
   - 避免手指接触运动部件
   - 固定好工件和刀具

2. 急停
   - 熟悉急停按钮位置
   - 保持紧急情况下可以快速触及

3. 环境
   - 保持工作区域整洁
   - 移除松散物品和衣物
```

### 操作安全

```
1. 启动前检查
   - 确认限位开关工作正常
   - 检查刀具安装牢固
   - 确认工件夹持稳固

2. 运行监控
   - 加工过程中保持观察
   - 注意异常声音或气味
   - 随时准备按急停

3. 停机后
   - 先关闭主轴
   - 等待运动停止
   - 关闭控制电源
```

## 进阶应用

### 高级 G 代码功能

#### 坐标系设置

```gcode
; 设置工件坐标系
G10 L2 P1 X0 Y0 Z0    ; 设置坐标系 1
G10 L2 P2 X100 Y100 Z0 ; 设置坐标系 2

; 选择工件坐标系
G54  ; 使用坐标系 1
G55  ; 使用坐标系 2
G56  ; 使用坐标系 3
```

#### 暂停和冷却液

```gcode
; 暂停 (M0 后需要手动恢复)
G0 X10 Y10
M0
G0 X20 Y20

; 可选暂停 (M1, 需软件支持)
M1

; 保持 (暂停但保持状态)
M0  ; 暂停
G4 P5000  ; 延时 5 秒
```

### 自动对刀

```gcode
; 使用探针自动设置 Z 轴原点
G38.2 Z-10 F50    ; 探针向下移动
G38.3 Z10 F20     ; 抬起
G92 Z0            ; 设置 Z=0
```
