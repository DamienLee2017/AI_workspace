## 硬件准备

### 必需的电子元件

| 元件 | 规格 | 数量 | 备注 |
|------|------|------|------|
| ESP32-S3 开发板 | ESP32-S3-DevKitC-1 或兼容板 | 1 | 推荐带 PSRAM 版本 |
| 步进电机驱动板 | TMC2209 / TMC2208 / A4988 / DRV8825 | 3 | 推荐 TMC2209 静音驱动 |
| 步进电机 | NEMA 17 (1.8° / 2.5A) | 3 | X/Y/Z 轴 |
| 开关电源 | 24V 10A 或 12V 8A | 1 | 根据驱动板和电机选择 |
| CNC 屏蔽板 | GRBL 兼容 | 1 | 可选，提供保护 |
| 限位开关 | 机械式 (NC 常闭) | 6 | 每轴 2 个 |
| 散热片 | 适合驱动芯片 | 3 | 必须 |
| 风扇 | 40mm x 40mm 12V/24V | 1-2 | 散热用 |

### 可选元件

| 元件 | 用途 | 备注 |
|------|------|------|
| 主轴电机 | 雕刻/铣削 | 功率根据需求 |
| 主轴驱动板 | PWM 速度控制 | 支持 0-10V PWM 输入 |
| 继电器模块 | 主轴启停控制 | 5V 继电器 |
| 接触器 | 总电源控制 | 急停用 |
| 急停按钮 | 紧急停止 | 带保护罩 |
| LED 指示灯 | 状态显示 | 电源/运行/报警 |

### 引脚详细定义

| 功能 | GPIO | 说明 | 连接目标 |
|------|------|------|----------|
| X Step | GPIO 1 | X 轴步进脉冲 | 驱动板 STEP |
| X Dir | GPIO 5 | X 轴方向 | 驱动板 DIR |
| Y Step | GPIO 8 | Y 轴步进脉冲 | 驱动板 STEP |
| Y Dir | GPIO 12 | Y 轴方向 | 驱动板 DIR |
| Z Step | GPIO 17 | Z 轴步进脉冲 | 驱动板 STEP |
| Z Dir | GPIO 21 | Z 轴方向 | 驱动板 DIR |
| Enable | GPIO 7 | 电机使能 (低电平有效) | 所有驱动板 ENABLE |
| X Limit Min | GPIO 40 | X 轴负限位 | 限位开关 |
| X Limit Max | GPIO 41 | X 轴正限位 | 限位开关 |
| Y Limit Min | GPIO 42 | Y 轴负限位 | 限位开关 |
| Y Limit Max | GPIO 43 | Y 轴正限位 | 限位开关 |
| Z Limit Min | GPIO 44 | Z 轴负限位 | 限位开关 |
| Z Limit Max | GPIO 45 | Z 轴正限位 | 限位开关 |
| Spindle PWM | GPIO 46 | 主轴速度 PWM | 主轴驱动 PWM 输入 |
| Probe | GPIO 47 | 触碰探针 | 探针信号线 |
| UART TX | GPIO 43 | 调试串口 TX | USB TTL 转换器 |
| UART RX | GPIO 44 | 调试串口 RX | USB TTL 转换器 |

### GPIO 使用限制

⚠️ **注意事项**：
- GPIO 0-21 可用于步进电机控制
- GPIO 40-47 仅支持输入功能，不能用于输出
- 不要使用内置 FLASH 相关的引脚 (GPIO 46, 47 在某些配置下)
- 建议预留 GPIO 38-39 用于 I2C 扩展

### 电源连接说明

| 电源类型 | 连接位置 | 说明 | 线材规格 |
|----------|----------|------|----------|
| 主电源 | 开关电源 L/N 输入 | AC 市电，注意接地 | 0.75mm² |
| 24V 输出+ | 驱动板 VMOT+ | 所有驱动板并联 | 1.5mm² |
| 24V 输出- | 驱动板 GND | 电源公共端 | 1.5mm² |
| ESP32 供电 | USB 或 5V 稳压器 | 建议独立供电 | 0.75mm² |

### 电源注意事项

⚠️ **重要安全提醒**：
- 首次接线前务必确认电源已关闭
- 电源输出电压必须与设备额定电压匹配
- 驱动板 VMOT 需要大容量电容 (建议 100μF 以上)
- 主电源线和信号线应分开走线，减少干扰
- 建议在电源输入端添加保险丝或断路器

## 驱动板连接

### TMC2209 驱动板连接 (推荐)

TMC2209 是静音步进驱动芯片，推荐用于 CNC 应用。

```
TMC2209 驱动板引脚布局:

    ┌─────────────────────────────────────────┐
    │  VMOT        GND         STEP    EN    │
    │   (+)        (-)         (+)     (+)  │
    │    │          │           │        │   │
    │    │          │           │        │   │
    │    ▼          ▼           ▼        ▼   │
    │ ┌────┐     ┌────┐     ┌────┐   ┌────┐ │
    │ │    │     │    │     │    │   │    │ │
    │ └────┘     └────┘     └────┘   └────┘ │
    │                                         │
    │  2B        2A         GND    MS1  MS2 │
    │    │        │          │        │    │ │
    │    ▼        ▼          ▼        ▼    ▼ │
    │ ┌────┐     ┌────┐     ┌────┐   ┌────┐ │
    │ │    │     │    │     │    │   │    │ │
    │ └────┘     └────┘     └────┘   └────┘ │
    │    │        │                        │
    │    ▼        ▼                        │
    │  电机 A 相  电机 B 相                 │
    └─────────────────────────────────────┘
```

### 连接步骤

1. **电机连接**：
   - 将步进电机的 A+、A- 连接到驱动板的 2A、2B
   - 将步进电机的 B+、B- 连接到驱动板的 1A、1B
   - 注意电机接线顺序，错误可能导致电机震动但不转动

2. **电源连接**：
   - VMOT 连接 24V/12V 正极
   - GND 连接电源负极
   - 建议在 VMOT 和 GND 之间添加 100μF 电容

3. **信号连接**：
   - STEP 引脚连接到 ESP32 对应轴的步进引脚
   - DIR 引脚连接到 ESP32 对应轴的方向引脚
   - EN 引脚连接到 ESP32 的 Enable 引脚
   - 所有信号 GND 连接到 ESP32 的 GND

### 驱动板电流配置

⚠️ **重要**：驱动板必须正确配置电流，否则可能导致电机发热严重或失步。

| 驱动板型号 | 参考电压 (VREF) | 最大电流 | 设置方法 |
|-----------|----------------|---------|----------|
| TMC2209 | 1.0-1.5V | 2.0A (外部) | 电位器调整 |
| DRV8825 | 0.5-1.0V | 2.5A (带散热) | 电位器调整 |
| A4988 | 0.3-0.6V | 2.0A (带散热) | 电位器调整 |

**VREF 计算公式**：
- DRV8825: I_max = VREF × 2
- A4988: I_max = VREF × 2.5

例如：NEMA 17 电机额定电流 1.5A
- DRV8825: VREF = 1.5 / 2 = 0.75V
- A4988: VREF = 1.5 / 2.5 = 0.6V

## 限位开关连接

### 限位开关类型选择

| 类型 | 优点 | 缺点 | 推荐场景 |
|------|------|------|----------|
| 机械开关 (NC) | 可靠、便宜 | 有机械磨损 | 通用场景 |
| 机械开关 (NO) | 简单接线 | 需要上拉电阻 | 简单项目 |
| 光学开关 | 无接触、寿命长 | 需要调整位置 | 高速应用 |
| 磁感应开关 | 非接触、耐用 | 需要磁铁 | 恶劣环境 |

### 限位开关接线 (NC 常闭型)

推荐使用 **NC (常闭)** 型限位开关，因为：
- 断线时系统会自动触发保护
- 比 NO 型更安全可靠

```
NC 型限位开关接线:

    ESP32 GPIO (输入) ──┬──► 限位开关 (NC) ──┬──► GND
                       │                    │
                       │    ┌───┐          │
                       └───►│10K│──────────┘
                            └───┘
                           (上拉)
```

## 主轴控制

### 主轴 PWM 控制

ESP32-S3 可以通过 PWM 信号控制主轴转速。

```
主轴控制连接图:

    ESP32-S3                主轴驱动/继电器
    GPIO 46 (PWM) ──────────► PWM 输入
         │                        │
         │                        ▼
         │                  ┌─────────────┐
         │                  │   主轴电机   │
         │                  │   (或调速器) │
         │                  └─────────────┘
         │
         ▼
    GND ──────────────────────► GND 公共端
```

### 主轴控制模式

| 模式 | 说明 | 实现方式 |
|------|------|----------|
| 继电器开关 | 仅启停，无调速 | GPIO 控制继电器 |
| PWM 调速 | 0-100% 速度可调 | GPIO PWM 输出 |
| 0-10V 模拟 | 工业标准调速 | PWM + RC 滤波 + 运放 |

### PWM 调速电路 (可选)

## 常见问题

### Q1: 步进电机抖动但不转动

**可能原因**：
1. 电机接线顺序错误
2. 驱动板电流设置过低
3. 电源电压不足

**解决方法**：
1. 交换电机 A 相或 B 相的两根线
2. 调整驱动板 VREF 电压
3. 检查电源输出功率

### Q2: 限位开关不断触发

**可能原因**：
1. 限位开关类型不匹配 (NC/NO)
2. 接线错误
3. 上拉/下拉电阻配置错误

**解决方法**：
1. 检查限位开关类型，NC 型需要下拉
2. 检查接线是否正确
3. 在代码中检查引脚模式配置

### Q3: 步进电机发热严重

**可能原因**：
1. 驱动板电流设置过高
2. 散热不足
3. 电机长时间静止但保持通电

**解决方法**：
1. 降低驱动板 VREF 电压 (但不能低于电机额定电流)
2. 添加散热片和风扇
3. 空闲时禁用电机 (ESP32 ENABLE 信号)

### Q4: 运动方向与预期相反

**可能原因**：
1. 电机接线反相
2. 方向信号反转

**解决方法**：
1. 交换电机 A 相和 B 相
2. 或在 Config.h 中设置 `DIR_INVERT`

### Q5: 串口通信不稳定

**可能原因**：
1. USB 线质量差
2. 电源干扰
3. 波特率不匹配

**解决方法**：
1. 使用优质的 USB 数据线
2. 将信号线远离电源线
3. 检查代码中波特率设置

## 接线安全提示

⚠️ **重要安全提醒**：

1. **断电操作**：在进行任何接线操作前，必须关闭电源并等待 30 秒让电容放电

2. **防止短路**：接线时注意正负极不要短路，特别是使用螺丝刀等金属工具时

3. **绝缘保护**：裸露的导线必须使用热缩管或电工胶带绝缘

4. **接地可靠**：设备外壳必须可靠接地，防止触电

5. **散热处理**：驱动板必须安装散热片，空闲时注意通风

6. **急停装置**：必须安装急停按钮，并确保能够快速切断电源

7. **首次测试**：首次上电时，用手靠近机器，准备随时断电
